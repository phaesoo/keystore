# Shield

[![Build Status](https://travis-ci.com/phaesoo/shield.svg?branch=main)](https://travis-ci.com/phaesoo/shield)

Shield is a standalone micro service for authentication, access control.

# Getting started

## Install dependencies
```bash
make deps
```

## Build
```bash
make main
```

## Run
```bash
./main
```

# Testing
Unit test
```
make test
```

Integration test
```
make test-ci
```
